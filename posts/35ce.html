日常

TIM

介绍：
TIM，QQ办公简洁版，是一款专注于团队办公协作的跨平台沟通工具。登录后，你的QQ好友都在，提供云文件、在线文档、邮件、日程、收藏等好用的办公功能，界面简洁清晰，QQ好友和消息无缝同步。

特点：

  • QQ的办公版
  • 简洁且无广告
  • 功能齐全，待办事项、日程、在线收发邮件等。

官网: https://tim.qq.com

Chrome

介绍：
Chrome无疑是浏览器界的NO.1；Chrome的整体发展目标是提升稳定性、速度和安全性，并创造出简单且有效率的用户界面。

据StatCounter统计，截至2020年1月，Chrome在全球桌面浏览器中有69.89%的占有率。

特点：

  • 无捆绑主页
  • 浏览速度快以及非常灵活的标签
  • 强大的可拓展性，即丰富的扩展程序商店
  • 更好的稳定性和安全性

官网: https://www.google.com/intl/zh-CN/chrome

华为云空间

介绍：
华为云空间是华为手机上用于备份程序数据和文件的一个官方工具。不同华为设备间登录同一账号可同步联系人、相册和备忘录等，不同设备的备份数据互不干扰，连接WiFi时一个星期会自动备份一次设备数据，一个设备有最近三次备份存档可用于恢复还原。

目前除了移动端集成到系统内之外，仅Win桌面端有客户端程序。若另购买空间（免费版除非不备份才可能有空间当云盘用）可当作私有云盘使用，可在手机、平板、电脑间相互同步传递文件。

官网: https://cloud.huawei.com

Sublime Text 3

介绍：
Sublime Text 3 是一款具有代码高亮、语法提示、自动完成且反应快速的代码编辑器软件，不仅具有华丽的界面，还支持插件扩展机制。

通过安装 Markdowd Editing 插件即可开启 Markdowd 写作。

首次安装需要修改配置才能符合我们的使用习惯。
配置教程可参考：https://www.jianshu.com/p/aab80e642960

官网： https://www.sublimetext.com/3

本站下载： https://senorui.lanzoui.com/iMtY0hzvvwb

Windows Terminal

介绍：
Windows Terminal 是一个全新的、流行的、功能强大的命令行终端工具，一个面向命令行工具和 shell（如命令提示符、PowerShell 和适用于 Linux 的 Windows 子系统 (WSL)）用户的新式终端应用程序。
它的主要功能包括多个选项卡、窗格、Unicode 和 UTF-8 字符支持、GPU 加速文本呈现引擎，你还可用它来创建你自己的主题并自定义文本、颜色、背景和快捷方式。

更多用法见： https://senorui.top/posts/1fd4.html

Github地址： https://github.com/microsoft/terminal
Microsotf Store：https://www.microsoft.com/zh-cn/p/windows-terminal/9n0dx20hk701

Office Tool Plus

介绍：
此工具可一键部署、激活以及移除Office，并支持选择性的安装所需的Office组件，可选择微软Office部署工具安装，所以无需担心安全问题。

比如我只想安装Word、Excel、PowerPoint。

Github： https://github.com/YerongAI/Office-Tool
官网： https://otp.landian.vip/zh-cn

Zoho Maill Desktop

介绍：
Zoho Mail桌面客户端，方便收发管理个人邮件。

目前个人用的是Zoho免费版域名邮箱，支持创建5个邮箱用户，每个用户5G空间，完全够普通人使用了。

官网: https://www.zoho.com/mail/desktop/

娱乐

HMCL启动器

介绍：
一款强大的MC启动器，支持灵活安装MC版本以及配置材质包、光影和Mod等。

MC正版用户，虽然入坑较晚但是空闲时间还是会玩玩。

官方： https://github.com/huanghongxun/HMCL

Potplayer

个人认为 PotPlayer 可以说是当今 Windows 平台上功能最强大的多媒体播放器，没有之一。

如果你拥有直播源还可以用它看电视直播。

特点：

  • 功能强大，设置齐全
  • 体验好，不流氓
  • 性能强劲，播放强大
  • 视频/音频格式支持非常全面
  • 自定义选项丰富，可自定义设置
  • 精美的界面，可自定义皮肤

官网： https://potplayer.daum.net （需要代理）
本站下载： https://senorui.lanzoui.com/i7m5vhzuyne

火绒安全是一款专业尽职、极致低调、只为巩固安全防线的安全软件。其最强大且实用的弹窗拦截以及火绒剑和自定义防护引擎。

官网： https://www.huorong.cn/person5.html

Geek

介绍：
Geek Uninstaller 是一个高效、快速、小巧、便携、免费的卸载软件。

特点：

  • 强制删除顽固应用
  • 清除已卸载应用安装文件和注册表值
  • 易于使用的用户界面
  • 无需安装，即点即用
  • 可卸载Windows Store的应用

LockHunter

LockHunter中文名称：解锁猎人，它能对占用文件进行解锁，还能强制删除占用文件，对付那些流氓程序的卸载残留是非常有效果的。

noMeiryoUI

介绍：
noMeiryoUI是Windows 8/8.1/10 上的系统字体设置工具，随意设置自己想要的字体，搭配 MacType效果更佳。

Github: https://github.com/Tatsu-syo/noMeiryoUI

MacType

介绍：
MacType 是基于一个 GDI++ 开源项目开发的字体渲染软件，MacType 给 Windows 用户带来了优良的字体显示效果，也使得我们能够「热替换」系统默认字体而无需修改系统文件。

其搭配自己喜欢的字体效果更佳。

Github: https://github.com/snowie2000/MacType
官网： https://www.mactype.net (较旧的发行版本)

O＆O ShutUp10*

介绍：
O＆O ShutUp10 一个小实用程序，可对隐私安全进行设置，并允许用户在 Windows 10中禁用一些系统组件。

特点：

  • 支持自定义调整系统安全设置项目
  • 支持开启或关闭系统隐私保护项目
  • 支持控制系统中的服务项
  • 支持控制windows系统的更新项目，我们可自定义打开或关闭windows系统更新功能
  • 支持开启或关闭向微软发送系统诊断信息与用户数据
  • 支持创建系统设置还原点，可在我们对系统设置修改前备份系统默认设置

注意：务必根据软件提示创建系统还原点，并谨慎设置每一个选项！

官网： https://www.oo-software.com/en/shutup10

实用工具

IDM*

介绍：
Internet Download Manager（IDM） 是一种http下载工具，可使下载速度提高 5 倍，可以全面恢复、重新启动中断的下载。简单的图形用户界面， IDM 用户友好且易于使用。IDM有一个智能下载逻辑加速器，具有智能动态文件分段和安全的多部分下载技术，加快您的下载。IDM在下载过程中动态下载文件，实现最佳的加速性能。

特点：

  • IDM支持代理服务器、ftp 和 http 协议、音频和视频内容下载
  • IDM无缝集成到微软Chrome、IE、Edge、opera、Mozilla 火狐所有热门浏览器，可自动处理您的下载
  • IDM有一个智能下载逻辑加速器，具有智能动态文件分割和安全的多部分下载技术，加快您的下载
  • IDM可以在设定的时间自动联网自动下载你想要的文件，然后挂断，甚至完成后自动关闭您的计算机
  • 您还可以拖放文件，或使用命令行的IDM

官网： https://www.internetdownloadmanager.com/

Bandizip

介绍：
Bandizip 是一个强大的多功能解压缩文件管理工具，它提供了高速的解压缩速度和其他各种实用功能。 这是一个免费的软件，其付费版本更是能够使用更多实用功能。

特点：

  • 支持的操作系统: Windows Vista/7/8/8.1/10 (32bit/64bit/ARM64)
  • 集压缩/解压/浏览/编辑为一体的压缩包管理器
  • 可以提取包括 RAR/ISO/7Z/ZIP 在内的 30 多种格式
  • 可以创建拥有密码和压缩分卷的压缩包
  • 支持调用更多的核心进行高速的解压缩

注意： 推荐下载 V6.29的标准版，Bandizip在标准版的 V7.0以后加入了广告。

官网： https://cn.bandisoft.com/bandizip/old/6
本站下载： https://senorui.lanzoui.com/iaGq1i0ij2h

MouseInc

介绍：
MouseInc是一个小巧好用的全局鼠标手势。便携化设计，单执行文件，无运行库依赖，无需安装即可运行，首次运行时会在当前目录释放MouseInc.json配置文件。

作者郑重承诺永久免费，无广告，不扫描硬盘，不上传隐私。

官网： https://shuax.com/project/mouseinc

Imagine

介绍：
Imagine是一款可用于批量压缩PNG和JPEG的桌面应用程序，其具有现代友好的UI。并能将指定的单个或多个图片文件压缩到需要的大小并保证图片质量不受损。

可以说是写博客必备图片压缩客户端！

官网： https://github.com/meowtec/Imagine

WinSCP

介绍：
WinSCP 是一个 Windows 环境下使用的 SSH 的开源图形化 SFTP 客户端。同时支持 SCP 协议。它的主要功能是在本地与远程计算机间安全地复制文件，并且可以直接编辑文件。

本站是部署到服务器上，但是GIT同步一直没折腾好，目前就通过此软件将静态文件上传到网站根目录下的。

有兴趣的可以移步查看部署过程： https://senorui.top/posts/8779.html

特点：

  • 图形用户界面
  • 内置文本编辑器
  • 可选地存储会话信息
  • 与 Windows 完美集成(拖拽, URL, 快捷方式)
  • 支持基于 SSH-1、SSH-2 的 SFTP 和 SCP 协议
  • 支持批处理脚本和命令行方式
  • 多种半自动、自动的目录同步方式
  • 支持 SSH 密码、键盘交互、公钥和 Kerberos(GSS) 验证
  • 通过与 Pageant(PuTTY Agent)集成支持各种类型公钥验证
  • 提供 Windows Explorer 与 Norton Commander 界面
  • 可将设置存在配置文件中而非注册表中，适合在移动介质上操作

官网： https://winscp.net/eng/index.php

Bulk Rename Utility

介绍：
Bulk Rename Utility是一款文件重命名工具，用它你可以批量给文件或文件夹重命名，随时更名随时预览。

你可以批量给文件或文件夹名称添加前缀或后缀，添加日期，自动编号，批量删除指定字符，批量插入指定字符到文件名中的指定位置，批量替换指定字符，批量删除数字、汉字等…更难能可贵的是，它还支持文件扩展名的批量更改，保存更名后的文件到新的位置。

特点：

  • 快速重命名多个文件，根据许多灵活的标准。
  • 多元化的重命名方式，且可删除或更改文件扩展名。
  • 在文件重命名前可以即时预览。
  • 重命名MP3文件的ID3标签(又名MP3 ID3标签重命名)。
  • 更改文件的创建日期和修改时间等。

官网： https://www.bulkrenameutility.co.uk
