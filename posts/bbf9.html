初窥 Linux Mint

 2021.01.20  Senorui   烟岚云岫  字数 1.2k  耗时 4分  热度 ℃

Linux Mint 由Linux Mint Team团队于2006年开始发行的一份基于Debian和Ubuntu的Linux发行版。

它诞生的目的是为家庭用户和企业提供一个免费的，易用的，舒适而优雅的桌面操作系统。

此文简单的讲述一下如何在实体机上安装 Linux Mint 操作系统。

准备工作

所需工具：

  • 电脑一台
  • U盘一枚
  • Linux Mint 系统镜像
  • 镜像写入工具

这里所述为在windows上实现以上准备工作。

去 Linux Mint 官网下载好系统镜像，Linux Mint 有三个EDITION可选择，其实就是个桌面版本不同而已，我这里选择 Cinnamon。

① 下载启动盘制作工具 rufus 并根据官网使用说明制作启动盘即可。

② 或者直接使用 Ventoy 制作，直接将ISO文件存U盘即可。

安装系统

将创建好ISO镜像的U盘插入电脑并重启电脑，当看到开机画面时按F2进入BIOS界面，在Boot选项中启动盘并确定，随后即可进入LinuxLive。

然后点击桌面上的 Install LinuxMint 即可开始安装，这里需要考虑整个磁盘安装Linux Mint 还是进行分区安装，若分区安装可打开 Gparted 调整分区，分出一个EXT4分区以便安装Linux（记下分区编号如sda2），再分出一个SWAP分区。

若整盘安装Linux Mint则在安装选项中选择删除磁盘所有数据并安装系统，若在分区中安装则需根据刚刚分的区去自己设置，设置好后点击立即安装即可。

随后需要输入用户名、用户密码等即可正式开始安装。

PS：若整磁盘安装 Linux Mint ，在开始前记得挂载磁盘，不然重启会先进入grub界面再进入系统。

配置系统

简单讲一下系统的重要配置项，软件源、系统语言、输入法的配置。

配置软件源

在 系统设置→系统管理→软件源→镜像 中将主要和基础源修改为国内源，根据连接速度选择最快的即可。

配置语言

在 系统设置→首选项→语言 中将所有项全设置为 Chinese,China 并应用到整个系统中，记得在预言支持中查看语言是否完全安装，否则需要点击一下安装语言包按钮，设置完毕重启即可。

若仍然有少部分界面是英文，则执行以下命令并重启。

配置输入法

在 Linux Mint上我们使用搜狗输入法，先去下载搜狗输入法安装包。

下载地址：https://pinyin.sogou.com/linux/?r=pinyin

然后在 在 系统设置→首选项→输入法→简体中文 中根据提示安装fcitx并注销重新登录系统，再安装下载的搜狗输入法。

最后在系统托盘中右键打开配置界面，设置搜狗输入法为第一个输入法即可，此后输入法使用与WIN无异。

安装软件

大多数常用软件均可在自带的 软件管理器 中安装，少数需要寻找软件包来安装。

下面将述需要下载安装包的Google Chrome和基于Wine的QQ、TIM、微信、迅雷和WinRAR等的安装。

Google Chrome

先在官网下载安装包：https://www.google.cn/intl/zh-CN/chrome/

在下载的.deb包上右键选择 使用 GDebi软件包安装程序 打开即可安装。

Wine软件
