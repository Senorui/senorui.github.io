常用软件

QQ&TIM

QQ是腾讯公司推出的一款基于互联网的即时通信软件，此全新体验版QQ下载 。而TIM是QQ的办公版，支持多人在线编辑Word、Excel文档等，更加适合办公用户使用。

QQ官网：https://im.qq.com/pcqq/index.shtml
TIM官网：https://office.qq.com
Tim下载*：https://www.123pan.com/s/HQeA-Hr1Sh

微信

微信是腾讯公司于2011年1月21日推出的一个为智能终端提供即时通讯服务的应用程序。

官网：https://weixin.qq.com
微信下载*：http://ct.ghpym.com/d/7369060-41603217-5866d1

Firefox

Firefox中文俗称“火狐”，是一个由Mozilla开发的自由及开放源代码的网页浏览器，使用Gecko排版引擎，支持多种操作系统。

官网：https://www.mozilla.org/zh-CN/firefox/new

Chrome

Chrome无疑是浏览器界的NO.1。

官网：https://www.google.com/intl/zh-CN/chrome

VLC

VLC Media Player 是一款可播放大多数格式，而无需安装编解码器包的媒体播放器。

官网：https://www.videolan.org/vlc

Bandizip

Bandizip 是一款压缩软件，它支持Zip、7-Zip 和 RAR 以及其它压缩格式。它拥有非常快速的压缩和解压缩的算法，适用于多核心压缩、快速拖放、高速压缩等功能。

官网：https://www.bandisoft.com/bandizip/old/6

Honeyview

Honeyview 是一款快速的图片查看器

官网：https://cn.bandisoft.com/honeyview

Sublimetext

Sublime Text是一个代码编辑器，也 是HTML和散文先进的文本编辑器 。

官网：https://www.sublimetext.com/download

Imagine

Imagine是一款可用于批量压缩PNG和JPEG的桌面应用程序，其具有现代友好的UI。并能将指定的单个或多个图片文件压缩到需要的大小并保证图片质量不受损。

官网： https://github.com/meowtec/Imagine

Waifu2x-caffe

waifu2x-caffe汉化版是一款免费的图片降噪软件，可以显著提升图片放大拉伸后的清晰度

官网：https://github.com/lltcggie/waifu2x-caffe/releases

Watt Toolkit

Watt Toolkit是一个开源跨平台的多功能 Steam 工具箱，可使 Steam 社区、 GitHub 、谷歌验证码等国内难以访问的网页正常访问，以及账号切换、脚本配置、库存管理和自动挂卡等功能

官网：https://steampp.net

小白羊云盘

小白羊云盘是一个开源的客户端。

官网：https://github.com/gaozhangmin/aliyunpan/releases

Adobe Acrobat Pro

Adobe Acrobat Pro可以使商业人士能够可靠地创建、合并和控制 Adobe PDF 文档，以及编辑PDF文档。

下载*：https://www.123pan.com/s/rha9-FOi63.html

实用工具

Nodejs&GIT

Node.js是一个基于Chrome V8引擎的JavaScript运行环境，它让JavaScript成为与PHP、Python、Perl、Ruby等服务端语言平起平坐的脚本语言；Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件，是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。

Nodejs官网：https://nodejs.org/zh-cn
Git官网：https://git-scm.com/download/win

Geek Uninstaller

Geek Uninstaller为一款小巧的卸载神器。

官网：https://geekuninstaller.com

LockHunter

LockHunter中文名称：解锁猎人，它能对占用文件进行解锁，还能强制删除占用文件，对付那些流氓程序的卸载残留是非常有效果的。

官网：https://lockhunter.com
汉化下载*： https://senorui.lanzoui.com/iDQevhzxuuf

OBS Studio

OBS Studio 是一款实时流媒体和屏幕录制软件，在直播领域使用的比较广泛。

官网：https://obsproject.com/

Office Tool Plus

Office Tool Plus是一个强大且实用的 Office 部署工具。

官网：https://otp.landian.vip/zh-cn
Github： https://github.com/YerongAI/Office-Tool

傲梅分区助手

傲梅分区助手是一款专业的无损分区工具。可在不丢失数据的情况下管理您的硬盘，完成分区相关的任何操作。

绿化下载*：https://www.123pan.com/s/HQeA-kD1Sh

CCleaner

CCleaner是一款系统优化和隐私保护工具，可用来清除Windows系统不再使用的垃圾文件，以及清除使用者的上网记录。

绿化下载*：https://www.123pan.com/s/HQeA-7n1Sh

FFRenamePro

FFRenamePro又名菲菲更名宝贝，是是一款优秀的批量更名软件和批量文件处理工具。

官网：http://www.ffhome.com/works/1406.html

Bulk Rename Utility

Bulk Rename Utility是一款文件重命名工具，用它你可以批量给文件或文件夹重命名，随时更名随时预览。你可以批量给文件或文件夹名称添加前缀或后缀，添加日期，自动编号，批量删除指定字符，批量插入指定字符到文件名中的指定位置，批量替换指定字符，批量删除数字、汉字等…更难能可贵的是，它还支持文件扩展名的批量更改，保存更名后的文件到新的位置。

官网： https://www.bulkrenameutility.co.uk

noMeiryoUI

NoMeiryoUI是一款非常棒的Windows系统字体修改工具。

官网：https://github.com/Tatsu-syo/noMeiryoUI/releases

MacType

MacType 是基于一个 GDI++ 开源项目开发的字体渲染软件，MacType 给 Windows 用户带来了优良的字体显示效果，也使得我们能够「热替换」系统默认字体而无需修改系统文件。

Github: https://github.com/snowie2000/MacType
官网： https://www.mactype.net (较旧的发行版本)

O＆O ShutUp10

O＆O ShutUp10 一个小实用程序，可对隐私安全进行设置，并允许用户在 Windows 10中禁用一些系统组件。

注意：务必根据软件提示创建系统还原点，并谨慎设置每一个选项！

官网： https://www.oo-software.com/en/shutup10

TranslucentTB

TranslucentTB是一款可以在 Windows 10/11 上使 Windows 任务栏半透明/透明的轻量小工具。

官网：https://github.com/TranslucentTB/TranslucentTB

WinSCP

WinSCP 是一个 Windows 环境下使用的 SSH 的开源图形化 SFTP 客户端。同时支持 SCP 协议。它的主要功能是在本地与远程计算机间安全地复制文件，并且可以直接编辑文件。

本站是部署到服务器上，但是GIT同步一直没折腾好，目前就通过此软件将静态文件上传到网站根目录下的。
